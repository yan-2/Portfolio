Project Overview and Challenges

The performance tool incorporates Joy-Con controllers via a web interface for gesture-based inputs and SuperCollider for sound synthesis.

The challenging part is that the connection between Joy-Cons is unstable as the controllers try to save energy. We use a classical observer pattern to wake up the controller when needed. Another issue is the connection between the supercollider and the web application. The middleware uses Node.js, and since the socket mode consumes a lot of resources, we use long-polling to conserve them.
