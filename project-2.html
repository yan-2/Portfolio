Project Overview and Challenges

The Content Management Service (CMS) has three main functions: configuring app information and behaviors, displaying analytics, and handling employee authentication.

Behavior configuration is particularly challenging as it involves multiple nested forms, which can cause rendering performance and state-passing issues. To address the rendering issue, we implemented the virtual DOM, which performs well even on mobile browsers. For state management, we applied Vuex to separate the different domain information.
